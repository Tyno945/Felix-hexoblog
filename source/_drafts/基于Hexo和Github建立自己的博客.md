欢迎来到风和博客！这是我的第一篇博客，在本篇博客中，我首先向大家介绍下如何建立一个自己个人的博客。我的博客网站是基于Hexo和Github Page搭建的，使用VScode编辑器软件和Markdown语法写博客。

**环境和工具准备**
- Windows10系统
- VS code编辑器
- [Node.js](http://nodejs.org/)
- [Git](http://git-scm.com/)
